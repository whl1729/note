# 申一帆的记忆方法

## 全脑记忆法十大原理

1. 图像原理。在记忆的时候要尽量出图，利用图像记忆效果会更好。
2. 色彩原理。想象的图像最好是有颜色的，这样记忆的效果更加深刻。
3. 动感原理。一方面，赋予物体动感。另一方面，加入自身的动作。
4. 联想原理。把要记的内容与我们已知的内容进行联系，帮助我们记忆。
5. 有趣原理。想象的故事或者场景中，加入一些新奇有趣、搞笑幽默、神奇好玩的元素，让印象更加深刻。
6. 夸张原理。两个小技巧，一个是变大变小，一个是变多变少。
7. 五感原理。充分调动我们的五种感官——视听味嗅触（眼耳口鼻手），来帮助我们记忆。
8. 关键原理。在记忆大量信息的过程中，要能有效提取关键信息。
9. 专注原理。在记忆的过程中，要提前定下目标，并且专注于此，这样效果更佳。
10. 重复原理。记忆之后，要在合理的时间节点重复复习，才能使记忆效果更好。

> 伍注：我需要加强运用的原理有：色彩、动感、五感。
